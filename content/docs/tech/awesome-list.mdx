A curated list of tools and resources, mostly open source.

## Analytics

- [Rudder](https://rudderstack.com/) (OSS) — Open-source Segment alternative
- [Umami](https://umami.is/) (OSS) — Website analytics alternative to Google Analytics
- [PostHog](https://posthog.com/) (OSS) — Product analytics, feature flags, session replay, and A/B testing

## Auth

- [Clerk](https://clerk.com/) — Drop-in authentication with prebuilt components
- [Auth.js](https://authjs.dev/) (OSS) — Authentication for Next.js and SvelteKit
- [WorkOS](https://workos.com/) — Enterprise SSO and user management

## Automation

- [Huginn](https://github.com/huginn/huginn) (OSS) — Create agents that monitor and act on your behalf
- [Airflow](https://airflow.apache.org/) (OSS) — Programmatically schedule and monitor workflows
- [Node-RED](https://nodered.org/) (OSS) — Low-code programming for event-driven applications
- [n8n](https://n8n.io/) (OSS) — Open source alternative for Zapier
- [Seliom](https://www.seliom.com/) — Automate human workflows
- [Pipedream](https://pipedream.com/) — The integration platform built for developers

## Browser Automation

- [Playwright](https://playwright.dev/) (OSS) — Browser automation for Chromium, Firefox, WebKit
- [Helium](https://github.com/mherrmann/selenium-python-helium) (OSS) — Selenium-Python variant
- [Selenium](https://www.selenium.dev/) (OSS) — Browser automation testing

## CMS

- [Contentful](https://www.contentful.com/) — Headless content management platform
- [Payload](https://payloadcms.com/) (OSS) — TypeScript-first headless CMS with Next.js integration

## Database & Backend

- [EasyDb](https://easydb.io/) — A one-click database, no server required
- [eSQLate](https://github.com/forbesmyester/esqlate) (OSS) — Build minimum viable admin panels with just SQL
- [Supabase](https://supabase.com/) (OSS) — Open-source Firebase alternative with Postgres, auth, and realtime
- [Staart](https://github.com/staart/api) (OSS) — Backend and payments starter for SaaS startups

## Design & CSS

- [Tailwind CSS](https://tailwindcss.com/) (OSS) — A utility-first CSS framework
- [Leon Sans](https://github.com/cmiscm/leonsans) (OSS) — Geometric sans-serif typeface
- [Base UI](https://base-ui.com/) (OSS) — Unstyled, accessible UI components from the MUI team
- [shadcn/ui](https://ui.shadcn.com/) (OSS) — Copy-paste component library built on Tailwind and Radix

## Development & Scripting

- [Intercooler.js](https://intercoolerjs.org/) (OSS) — AJAX with attributes
- [bash-lib](https://github.com/aks/bash-lib) (OSS) — Bash utility methods and tools
- [Snowpack](https://www.snowpack.dev/) (OSS) — Build a web application without a bundler

## Ecommerce

- [Stripe](https://stripe.com/) — Payments infrastructure for the internet
- [Shopify](https://www.shopify.com/) — The dominant hosted ecommerce platform
- [Medusa](https://medusajs.com/) (OSS) — Open-source headless commerce engine
- [Vendure](https://www.vendure.io/) (OSS) — TypeScript headless commerce framework built on Node.js

## Email

- [Resend](https://resend.com/) — Modern transactional email API
- [React Email](https://react.email/) (OSS) — Build emails with React components
- [Postmark](https://postmarkapp.com/) — Reliable transactional email delivery

## Forms & Data

- [React JSON Form](https://github.com/rjsf-team/react-jsonschema-form) (OSS) — Schema-driven form component
- [Uniforms](https://github.com/vazco/uniforms) (OSS) — Schema-driven form library
- [jExcel](https://github.com/jspreadsheet/ce) (OSS) — Create HTML spreadsheets

## Hosting & Static Sites

- [1MB](https://1mb.site/) — Free static website hosting and database
- [Fast](https://fast.io/) — Hosting for files, images, and apps
- [ipfs-deploy](https://github.com/ipfs-shipyard/ipfs-deploy) (OSS) — Deploy static websites to IPFS

## Image Processing

- [imgproxy](https://github.com/imgproxy/imgproxy) (OSS) — Resize and convert remote images
- [Thumbor](https://github.com/thumbor/thumbor) (OSS) — On-demand crop, resizing, and flipping

## Issue Tracking

- [Linear](https://linear.app/) — Issue tracking platform
- [Plane](https://plane.so/) (OSS) — Open-source project management alternative to Linear and Jira

## Media

- [soundcn](https://www.soundcn.xyz/) (OSS) — Free sound effects library for web applications and games

## Monitoring & Testing

- [Bird Eats Bug](https://birdeatsbug.com/) — Record and share bugs with console logs
- [Embrace](https://embrace.io/) — Application testing and monitoring tool
- [GoAccess](https://goaccess.io/) (OSS) — Analytics for access logs
- [Ciao](https://github.com/brotandgames/ciao) (OSS) — HTTP checks and tests monitoring

## NoCode & App Development

- [BravoApp](https://www.bravostudio.app/) — Turn Figma prototypes into native iOS & Android apps
- [Supernova](https://supernova.io/) — Convert design files to Flutter, iOS, Android, React Native apps
- [CoScreen](https://www.coscreen.co/) — Turns your secondary display into your joint team desktop
- [Draftbit](https://draftbit.com/) — Build apps from the browser
- [Autocode](https://autocode.com/) — Automatically generate API code for SaaS apps

## Search & Knowledge

- [Typesense](https://typesense.org/) (OSS) — Open-source alternative to Algolia
- [Tettra](https://tettra.com/) — Knowledge base for teams
- [Contextualise](https://github.com/brettkromkamp/contextualise) (OSS) — Manage your knowledge
- [Outline](https://www.getoutline.com/) (OSS) — Team wiki and knowledge base

## Security & Configuration

- [ssl-config](https://ssl-config.mozilla.org/) (OSS) — SSL configuration generator
- [SOPS](https://github.com/getsops/sops) (OSS) — Encrypt secrets in version control

## Specialized Tools

- [Webhook.site](https://webhook.site/) — Test webhooks and HTTP requests
- [Ollama](https://ollama.com/) (OSS) — Run large language models locally

## Tools & Infrastructure
